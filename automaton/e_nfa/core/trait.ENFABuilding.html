ENFABuilding is the trait assiociated to the ENFABuilder type. Each ENFABuilder should implement ENFABuilding trait.

ENFABuilder can generate some errors during the building stage. For instance, one could try to insert a transition with two different destination states.

Errors

If self contains a ENFAerror then each function should transfer this error.
