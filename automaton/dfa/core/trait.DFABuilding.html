DFABuilding is the trait assiociated to the DFABuilder type. Each DFABuilder should implement DFABuilding trait.

DFABuilder can generate some errors during the building stage. For instance, one could try to insert a transition with two different destination states.

Errors

If self contains a DFAerror then each function should transfer this error.
